# Configurar o Ambiente
## Editar public/resource/config.php
  - Alterar nome do Jogo (Linha 3)
  - Conferir dados de Acesso ao Banco de Dados MySQL (Linhas 13 à 26)

# Iniciar o Jogo 
## Pré Requisitos
 - php 8.x
 - MySQL
## Comandos para Inicialização
 - Usando um terminal (Power Shell / CMD / Terminal no Linux)
 - Acessar a pasta public do projeto (~/public)
 - Executar um servidor Web no local
>  php -S localhost:8080
## Acessar o site no navegador
>  <a href="http://localhost:8080">http://localhost:8080</a>
